# Hermes Agent 入门指南

欢迎来到 Hermes 101。这里用官方推荐路径，带你最快跑通 Hermes Agent。

## 前置要求

- Linux / macOS / WSL2 / Termux 任一环境
- 可用的模型 API Key（或本地模型）
- 终端基础操作能力

## 官方安装

```bash
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
```

安装完成后，编辑 `~/.hermes/config.yaml` 填入模型 API Key 和平台 token，然后运行 `hermes` 即可启动。
